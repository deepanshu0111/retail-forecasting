# 📈 Retail Demand Forecasting

## 📌 Overview

This project focuses on **predicting retail demand using historical data**.
It uses data processing and forecasting techniques to analyze sales trends and generate future demand predictions.

---

## 🚀 Features

* 📊 Data preprocessing and analysis
* 📈 Demand forecasting using historical data
* 🔄 Backend API for prediction handling
* ⚡ Lightweight and efficient implementation

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Machine Learning / Time Series concepts

---

## 📂 Project Structure

* `backend_api.py` → Handles prediction logic
* `frontend_api.py` → Interface for interacting with model
* `requirements.txt` → Dependencies
* `vercel.json` → Deployment configuration

---

## ⚙️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run backend:

```
python backend_api.py
```

3. Run frontend:

```
python frontend_api.py
```

---

## 📊 Output

The system processes retail data and generates demand predictions, which can be used for:

* Inventory planning
* Sales forecasting
* Business decision-making
